Welcome to SearchAlgorithms

This interactive tool demonstrates various Artificial Intelligence **Search Algorithms** by finding paths between cities on a simplified map of Romania.

Navigate to the **Maps** section to run a search query (e.g., 'Find path from Arad to Bucharest using A*').

Search Algorithm Breakdown

We visualize both **Uninformed (Blind)** and **Informed (Heuristic)** search strategies:

  • **Uninformed Search**: Algorithms that explore the search space without any domain knowledge or estimation of the path cost to the goal. This includes:
    • **Breadth-First Search (BFS)**: Explores nodes layer by layer, guaranteeing the shortest path in terms of steps.
    • **Depth-First Search (DFS)**: Explores as far as possible along each branch before backtracking, and is generally more memory-efficient.
    • **Uniform Cost Search (UCS)**: Expands the node with the lowest path cost from the start, guaranteeing the optimal (cheapest) path.
    • **Iterative Deepening DFS (IDDFS)**: Combines the completeness of BFS with the low memory usage of DFS.
  • **Informed Search**: Algorithms that use a **heuristic function** (an "educated guess" of the remaining cost to the goal) to guide the search. This includes:
    • **Greedy Best-First Search**: Prioritizes the node that appears closest to the goal (lowest heuristic value).
    • **A\* Search**: Combines the actual cost from the start ($g$) with the estimated cost to the goal ($h$), prioritizing the node with the lowest total estimated cost ($f = g + h$). It is guaranteed to find the **optimal path**.
